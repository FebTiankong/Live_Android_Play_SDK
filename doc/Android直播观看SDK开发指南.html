3.9 快速集成说明

完成上面的步骤后，基本上我们就能够观看到直播间的视频画面和文档画面了，也就基本完成了CC直播的核心功能的集成，即观看视频和观看文档的功能。后续在第4节会对CC直播SDK提供的其他的功能特性的使用做详细的说明。

4.功能使用

功能使用时相关的核心类：DWLiveListener（直播回调接口类），DWLive（直播核心类）。

4.1 聊天功能

聊天功能分为公共聊天和私聊，聊天时主动调用的方法在DWLive类，相关方法如下：

  • 发送公共聊天的方法：DWLive.sendPublicChatMsg(String publicChatMsg)。
  • 发送私聊的方法：DWLive.sendPrivateChatMsg(String toUserId, String msg)。

聊天功能的核心回调类为DWLiveListener，相关的回调方法如下：

  • 接收公共聊天对应的回调方法为：onPublicChatMessage(ChatMessage msg)，自己发的公共聊天数据和别人的公共聊天数据都是通过此接口回调回来。
  • 接收别人私聊我的回调方法：onPrivateChat(PrivateChatInfo info)。
  • 接收我私聊别人的回调方法：onPrivateChatSelf(PrivateChatInfo info)。
  • 自己被禁言后，发出的聊天数据会从onSilenceUserChatMessage(ChatMessage msg)方法回调回来，而不会通过onPublicChatMessage(ChatMessage msg)回调。
  • 客户端禁止聊天后，发出的聊天数据不会回调，会通过onInformation方法回调内容为：讲师暂停了文字聊天，请专心看直播吧。

4.2 问答功能

提问时主动调研的方法在DWLive类，相关方法如下：

  • 发送问题的方法：DWLive.sendQuestionMsg(String question)。

问答功能的核心回调类为DWLiveListener，相关的回调方法如下：

  • 接收问题的回调方法：onQuestion(Question question)，自己发送的问题和别人发送的问题都会从此接口回调回来。
  • 收到问题发布的ID的回调方法：onPublishQuestion(String questionId)，此方法可以配合Question的isPublish字段使用。
  • 收到回答的回调方法：onAnswer(Answer answer)。

4.3 签到功能

签到时主动调用的方法在DWLive类，相关方法如下：

  • 签到的方法：DWLive.sendRollCall()。

签到功能的核心回调类为DWLiveListener，相关的回调方法如下：

  • 收到签到事件的回调：onRollCall(int duration)。

4.4 抽奖功能

此功能为方法回调功能，相关类DWLiveListener，相关方法如下：

  • 收到开始抽奖事件的回调：onStartLottery(String lotteryId)。
  • 收到抽奖结果事件的回调：onLotteryResult(boolean isWin, String lotteryCode, String lotteryId, String winnerName)。
  • 收到结束抽奖事件的回调：onStopLottery(String lotteryId)。

4.5 公告功能

此功能为方法回调功能，相关类DWLiveListener，相关方法如下：

  • 收到公告的回调：onAnnouncement(boolean isRemove, String announcement)。

4.6 问卷功能

问卷功能主要为方法回调功能，相关类DWLiveListener，相关方法如下：

  • 收到问卷发布的回调：onQuestionnairePublish(QuestionnaireInfo info)。
  • 收到停止问卷的回调：onQuestionnaireStop(String questionnaireId)。
  • 收到第三方问卷发布的回调：onExeternalQuestionnairePublish(String title, String externalUrl)。
  • 收到问卷统计信息的回调：onQuestionnaireStatis(QuestionnaireStatisInfo info)。

同时提供了主动获取当前正在进行问卷的方法：

  • 主动获取问卷的方法：fetchQuestionnaire()。调用获取当前问卷的方法后，如果有正在进行的问卷，就会触发收到问卷发布回调方法onQuestionnairePublish。

4.7 广播功能

此功能为方法回调功能，相关类DWLiveListener，相关方法如下：

  • 收到广播消息的回调：onBroadcastMsg(String msg)
  • 收到历史广播消息的回调：onHistoryBroadcastMsg(ArrayList msgs)

4.8 答题卡功能（投票功能）

  • 收到开始答题（投票）的回调：onVoteStart(int voteCount, int VoteType)。
  • 收到结束答题（投票）的回调：onVoteStop()。
  • 收到答题（投票）结果统计的回调：onVoteResult(JSONObject jsonObject)。

4.9 连麦功能

连麦时主动调用的方法都在DWLive类，相关方法如下：

  • 设置连麦相关的参数：setRtcClientParameters(RtcClientListener rtcClientListener, SurfaceViewRenderer localRender, SurfaceViewRenderer remoteRender)
  • 开始视频连麦：startRtcConnect()
  • 开始音频连麦：startVoiceRTCConnect()
  • 断开连麦：disConnectSpeak()
  • 断开连麦申请：disConnectApplySpeak()
  • 关闭摄像头，释放资源：closeCamera()
  • 移除本地摄像头渲染：removeLocalRender()

连麦功能的核心回调类为RtcClient.RtcClientListener，相关的回调方法如下：

  • 回调当前是否能够连麦：onAllowSpeakStatus(boolean isAllowSpeak)
  • 回调主播端接通连麦,开始连麦：onEnterSpeak(String videoSize)
  • 回调连麦断开：onDisconnectSpeak()
  • 回调连麦异常：onSpeakError(Exception e)
  • 回调摄像头打开：onCameraOpen(int width, int height)

4.10 音频模式

此功能为主动调用功能，调用的方法在DWLive，相关方法如下：

  • 切换当前播放模式：changePlayMode(PlayMode playMode)

4.11 获取在线人数

此功能为方法回调功能，相关类DWLiveListener，相关方法如下：

  • 在线人数：onUserCountMessage(int count)

4.12 用户踢出

此方法为方法回调功能，相关类DWLiveListener，相关方法如下：

  • 用户被踢出房间的回调：onKickOut(int type)
